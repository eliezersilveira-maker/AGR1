# Agro Forte, Futuro Sustentável

Plataforma web moderna construída a partir dos argumentos estruturais e estatísticos de uma redação dissertativa-argumentativa humana sobre o equilíbrio entre a força do agronegócio e a preservação ambiental no Brasil.

## 🚀 Funcionalidades Incorporadas

- **Painel de Indicadores Interativos**: Exibição animada da participação do agronegócio no PIB nacional (24%).
- **Gráficos Dinâmicos**: Gráfico em barras gerado via *Chart.js* mostrando o panorama de metas sustentáveis e redução de emissões.
- **Simulador Lógico**: Mecanismo iterativo em JavaScript que demonstra as consequências práticas e mercadológicas entre o manejo antigo e a transição verde (ILPF).
- **Interface Responsiva & Dark Mode**: Layout fluido adaptável a dispositivos móveis com suporte nativo a tema escuro.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização baseada em variáveis e propriedades de Grid/Flexbox para responsividade.
- **JavaScript (ES6)**: Manipulação dinâmica de dados, animação de contadores e controle de tema.
- **Chart.js**: Biblioteca utilizada para a renderização limpa do gráfico estatístico.
- **FontAwesome**: Biblioteca de ícones vetoriais.
